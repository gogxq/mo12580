立即参加活动

活动时间

2012年10月8日-12月31日

活动对象

中国移动用户（下载并注册12580客户端用户）

活动内容

活动期间，客户端注册用户通过分享上传城市餐饮美食、休闲娱乐特色商户图片可获取相应积分，所得积分可使用户身份晋级，根据不同身份等级参与相应等级的活动抽奖。

奖项设置

用户根据不同身份等级参与相应级别的抽奖。详细见下

身份等级 累计积分数量 奖品
初学乍练 累计获得50分 获得2元话费
登堂入室 累计获得200分 可参加100元话费抽奖
自成一派 累计获得500分 可参加500元话费抽奖
已臻大成 累计获得1000分 可参加三星GALAXY SIII抽奖机会

活动规则

  1. 1.每个用户（手机号）上传图片，每成功上传1张有效图片可获10分。每用户每日最多可获50分，同张图片重复上传不得分。
  2. 2.图片上传路径：
    • （1）在客户端中选择一家想上传照片的商户，点击图片上的“拍照”按钮，拍照或从相册选取图片后提交（目前不支持客户端未录入的商家,上传路径根据用户使用客户端版本不同会略有差异）。
    • （2）点击客户端工具栏的”拍照“按钮，拍照或从相册选取图片后选择上传到商户，输入相应标题、类型即可。
  3. 3.用户可通过客户端会员区活动积分界面查询积分及身份等级，积分为即时显示。待审核图片积分将在图片审核通过后计入积分，图片上传后24小时内审核完毕，即可计入活动积分。
  4. 4.本次活动限 android（安卓）、ios（苹果）系统的终端，且安装2.5.9及以上版本的12580客户端用户。
  5. 5.图片内容要求参见：图片审核规则

中奖查询

中奖名单将在活动开始次月10日前公布，获奖名单将在客户端会员区公布（客户可登录12580客户端，进入“活动积分”查看本号码中奖信息），还可以拨打12580进行查询。

奖品发放

  1. 1.奖项的确认以参与手机号码是否为正常在网用户为准，若参与活动的手机号码销号、停机或处于非正常状态，则视为用户自动放弃奖品。如参与手机号码处于非正常状态无法获得任何奖项，将视为客户自动放弃奖品。
  2. 2.活动奖品将于中奖名单公布后二十个工作日完成领取通知及发放。奖品需客户自行承担奖品价值的税金，且需在规定领奖时间内（具体领奖时间以各省公司通知为准）领取，两个月内未领取奖品则视为自动放弃。

资费标准

  1. 1.客户登录12580客户端、上传图片或下载手机软件搜索信息按照GPRS正常资费收取。
  2. 2.接收12580客户端下发活动短信及中奖短信免通信费及信息费。
